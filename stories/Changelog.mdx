# Changelog

`@code-sherpas/pharos-react` follows semver enforced by Changesets. The
authoritative changelog lives in
[`CHANGELOG.md`](https://github.com/code-sherpas/pharos-react/blob/main/CHANGELOG.md)
at the repo root; this page summarises it for readers coming from Storybook.

## Semver policy

| Change                                       | Bump  |
| -------------------------------------------- | ----- |
| Rename or remove a component, variant, prop  | major |
| Change peer dependency range (react, tokens) | major |
| Add a new component, variant, or prop        | minor |
| Visual refinement, a11y fix, bug fix, perf   | patch |
| Internal build/tooling change                | patch |

## Latest releases

Once `0.1.0` is published, this page will render the most recent entries
from `CHANGELOG.md`. Until then, the story is: _scaffold in place, first
component `PharosHello` pending review_.
